COMMAND PROMPT


Engraçado E interessante!


CMD tambem conheçido como Linha De Comandos ou Command Prompt.

Com o CMD podemos fazer muitas ações desde Networking até oque é chamado de "Hacking" porem que a CMD do Windows não é nada util para esse tipo de Cyber. O CMD em sí é um intepertador de comandos OS/2 de sistemas baseados em Windows NT tambem usado em MS-DOS, sistemas Windows 9x e de shells usados em sistemas Unix.

Peer-2-Peer


Facil De Aprender E Relembra!r


Peer-2-Peer é um tipo de conexção comum em aplicações como o utorrent.

Peer-2-Peer pode porem ser uma conexção perigosa pois é uma conexção directa entre dois ou mais computadors onde ambos partilham e recebem ficheiros sendo facil aceder a um deles.

CPU-Multithreading


Complexo Para Quem Não Percebe.


CPU Multithreading é por sí a abilidade de usar varios threads na execução de uma aplicação.

Na Arquitetura de um computador um CPU capaz de multithreading permite o Utilizador expalhar os seus threads por varias aplicações assim dando mais threads as aplicações de maior prioridade e menos as que necessitam de menos recursos.
